# Homelab DevOps

## Описание проекта

Данный репозиторий содержит локальную DevOps-лабораторию для изучения и практики:

- Linux
- SSH
- Git
- Ansible
- Terraform / OpenTofu
- автоматизации развертывания инфраструктуры

Цель проекта — построить приближенную к реальной инфраструктуру с разделением ролей серверов и автоматизированным управлением.

## Серверы

| Сервер | Назначение |
|--------|-----------|
| bastion | Управляющий сервер для Ansible, Git и инфраструктурных задач |
| app01 | Сервер приложения |
| db01 | Сервер базы данных |
